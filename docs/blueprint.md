# **App Name**: Leidy's Portfolio

## Core Features:

- Profile Showcase: Display Leidy's professional profile with a summary of her experience and skills.
- Project Portfolio: Showcase featured projects with descriptions, technologies used, and links to GitHub repositories.
- Skills Overview: Provide a comprehensive overview of technical skills and tools.
- Services Listing: List the services offered, highlighting expertise in backend, frontend, and API development.
- Contact Form: Enable potential clients to contact Leidy directly through the portfolio website.
- AI-Powered Personalized Recommendation tool: Generates recommendations based on projects that a client may be interested in commissioning.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to convey professionalism and trust.
- Background color: Very light gray (#F5F5F5) for a clean, modern look.
- Accent color: Soft lavender (#C5CAE9) to add a touch of creativity and sophistication.
- Headline font: 'Poppins', a geometric sans-serif for a contemporary and clean aesthetic.
- Body font: 'PT Sans', a humanist sans-serif providing readability and modern feel. This pairs well with the Headline Font.
- Use clean, minimalist icons to represent skills, technologies, and services.
- Maintain a clean, well-organized layout with clear sections for easy navigation.
